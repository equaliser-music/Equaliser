NIP-05 Verification

NIP-05 lets NOSTR clients verify your identity using your domain name. Set your NIP-05 address in your Profile, then follow the steps below to configure your domain.

Set up domain verification
1

Download your verification file

This file proves you own this NOSTR identity. It must be hosted on your domain.

2

Upload to your website

Place the file so it is accessible at:

The file must be in a folder called .well-known at the root of your website.

3

Add CORS headers

NOSTR clients need permission to read this file from your domain. Add these headers to your web server config.

location /.well-known/nostr.json { add_header 'Access-Control-Allow-Origin' '*' always; add_header 'Access-Control-Allow-Methods' 'GET, OPTIONS' always; if ($request_method = 'OPTIONS') { return 204; } }
# In .htaccess or VirtualHost config <IfModule mod_headers.c> <Files "nostr.json"> Header set Access-Control-Allow-Origin "*" Header set Access-Control-Allow-Methods "GET, OPTIONS" </Files> </IfModule>
