NIP-05 Verification

A verified identifier that proves you own a domain. Shows a checkmark next to your name.

How to set up NIP-05:
  1. Choose a domain you own (e.g. yourdomain.com)
  2. Create a file at https://yourdomain.com/.well-known/nostr.json
  3. Add your public key to the JSON: {"names":{"you":"your-hex-pubkey"}}
  4. Enter you@yourdomain.com above and save
